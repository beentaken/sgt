Simon Tatham's Portable Puzzle Collection

Introduction

This page contains a collection of small computer programs which implement one-player puzzle games. All of them run natively on Unix (GTK), on Windows, and on Mac OS X.

I wrote this collection because I thought there should be more small desktop toys available: little games you can pop up in a window and play for two or three minutes while you take a break from whatever else you were doing. And I was also annoyed that every time I found a good game on (say) Unix, it wasn't available the next time I was sitting at a Windows machine, or vice versa; so I arranged that everything in my personal puzzle collection will happily run on both. When I find (or perhaps invent) further puzzle games that I like, they'll be added to this collection and will immediately be available on both platforms. And if anyone feels like writing any other front ends - Mac OS pre-10, PocketPC, or whatever it might be - then all the games in this framework will immediately become available on another platform as well.

The games

The actual games in this collection were mostly not my invention; I saw them elsewhere, and rewrote them in a form that was more convenient for me. I do not claim credit, in general, for inventing the rules of any of these puzzles; all I claim is authorship of the code.
Net

Reassemble the shuffled tiles into a fully connected network by rotating them back to their correct orientations.

Cube

Pick up the blue squares by rolling the cube over them. Try to get all six blue squares on the faces of the cube at once. Count your moves and try to do it in as few as possible. Then try it on a triangular grid with any of a tetrahedron, an octahedron or an icosahedron.

Fifteen

The good old "15-puzzle". Slide the tiles around to arrange them into numerical order.

Sixteen

Arrange the tiles into numerical order again, this time by rotating an entire row or column at a time within itself.

Twiddle

Arrange the tiles into numerical order yet again. This time your move is to rotate a square of four (or more!) tiles around each other.

Rectangles

Subdivide the grid into rectangles so that each rectangle contains only one number, and has area equal to that number.

Netslide

Combine two of these puzzles at once: use the sliding action of "Sixteen" to reassemble a grid in the style of "Net".

Pattern

Fill in a grid with a black and white pattern, given only the lengths of the continuous runs of black squares.

Solo

Fill in a grid full of digits so that each row, column and square block contains only one of each different digit.

Download

Here are Windows executables of the puzzle games in the collection. (On Windows only, the Net executable is called "netgame.exe" in order to avoid clashing with Windows's own "net.exe". The name of the game is still "Net" :-)

netgame.exe | cube.exe | fifteen.exe | sixteen.exe | twiddle.exe
rect.exe | netslide.exe | pattern.exe | solo.exe

Here is a Windows help file (and its accompanying contents file). If you install these in the same directory as the executable files, then each game should display a "Help" menu giving help about the game collection in general and that game in particular. You can also browse the same documentation online in HTML format.

puzzles.hlp | puzzles.cnt

Here is a Mac OS X disk image file, containing a single monolithic application called "Puzzles". You should be able to download and open the disk image, then drag the Puzzles application to wherever you feel like keeping it; it should be entirely self-contained. (However, it is also the first Mac OS X application I've ever written, so I can't make any guarantees! Let me know if you have any trouble with it and I'll try to fix it.)
